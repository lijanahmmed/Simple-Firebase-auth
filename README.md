# Simple Firebase Auth

A simple authentication system using **Firebase Authentication** with **Google** and **GitHub** login support. This project demonstrates how to integrate Firebase Auth into a React or web application for secure and easy authentication.

---

## Features

- Google Sign-In
- GitHub Sign-In
- Secure authentication using Firebase

---

## Technologies Used

- **Firebase Authentication** – For handling authentication
- **JavaScript / React** – Frontend framework
- **Tailwind CSS** (optional) – For styling
